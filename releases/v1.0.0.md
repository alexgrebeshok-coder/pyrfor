# CEOClaw v1.0.0

First public desktop release for the live-first CEOClaw product.

## What ships

- hosted web app as the source of truth;
- signed macOS DMG with the Tauri shell;
- live AI council runtime with replayable traces;
- public release hub with honest install-state badges;
- manifest-driven AI provider and connector layer;
- desktop hotkeys, window restore, and tray/menu behavior.

## Install

1. Download `CEOClaw_1.0.0_aarch64.dmg` from the release assets.
2. Open the DMG and drag CEOClaw into Applications.
3. Launch CEOClaw and sign in to the hosted live product.

## Known gaps

- The iPhone archive/build path still requires a machine with full Xcode installed.
- Public iPhone distribution is still pending the archive-ready build path.
