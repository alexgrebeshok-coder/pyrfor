# EVM Integration Dashboard

Краткий рабочий план для интеграции EVM-генератора в CEOClaw.

## Цель

Сделать отдельную точку входа для финансового моделирования, где пользователь сможет:

- собрать проектные параметры;
- сгенерировать Excel-модель EVM;
- увидеть графики PV / EV / AC;
- сравнить сценарии `Best / Base / Worst`;
- получить быстрый управленческий вывод по CPI, SPI, EAC и VAC.

## Что уже есть

- `scripts/generate_evm.py` - Python-скрипт генерации Excel.
- `components/analytics/evm/*` - существующие EVM-визуализации в интерфейсе.
- `lib/plan-fact/service.ts` - расчёты EVM внутри портфельной аналитики.

## Как встраивать

1. Добавить страницу запуска EVM-генератора в аналитике.
2. Подключить route, который вызывает `scripts/generate_evm.py`.
3. Показывать результат как скачиваемый `.xlsx` файл.
4. Связать генератор с портфельной аналитикой и карточками EVM.
5. Добавить отдельный документ в хаб `Документы` для этого потока.

## Следующий шаг

Подготовить пользовательский экран, где финансовый контур CEOClaw будет работать как сценарный калькулятор, а не как скрытая служебная утилита.
