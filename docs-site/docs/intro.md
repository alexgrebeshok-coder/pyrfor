# Welcome to Pyrfor

Pyrfor is a **local-first, governed** AI coding workspace: the desktop app (`Pyrfor.app`) and the open **`@pyrfor/engine`** runtime.

## What you get

- Universal Engine lifecycle: plan → research → execute → critique → postmortem → memory
- Permission ladder, budgets, and audit ledger
- MCP, blocks, and industrial reconciliation fixtures (see repo `fixtures/`)
