# OpenClaw Integration для CEOClaw Dashboard

Модуль для обработки голосовых и текстовых команд пользователя.

## Структура
- `integration.ts` - основной обработчик
- `intent-detection.ts` - логика парсинга (Regex)
- `commands/` - реализация конкретных команд
- `types.ts` - TypeScript типы

## Использование
```typescript
import { processCommand } from './lib/openclaw/integration';

const input = "Добавь задачу в ЧЭМК — согласовать СП";
const response = await processCommand(input);
// "Задача "согласовать СП" добавлена в проект ЧЭМК"
```

## Поддерживаемые команды
1. **Добавь задачу** ("Добавь задачу в [проект] — [задача]")
2. **Обнови бюджет** ("Обнови бюджет на [сумма]")
3. **Покажи статус** ("Покажи статус [проект]")
4. **Создай проект** ("Создай проект [название]")
5. **Назначь** ("Назначь [человек] на [задача]")
