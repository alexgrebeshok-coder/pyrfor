## 4. Конкретные Рекомендации по Исправлению

### 4.1 Скорректированный план (5 недель)

| Неделя | Copilot (Engine) | Cursor (Desktop) | FreeClaude |
|--------|-----------------|------------------|------------|
| **1** | M1 Substrate + API-контракт | Новый проект + Shadcn/UI + NavigationRail + ChatPanel | WebSocket Gateway Engine↔Desktop |
| **2** | M4 Memory Provider + ContextEngine | Agent Dashboard + Task Composer | Тесты Engine API |
| **3** | M6 Strategy Memory + Double-Loop | Approval Panel + Plan Viewer | Интеграция Desktop↔Engine |
| **4** | M7–M11 ToolForge + hardening | Полировка UI + InspectorPanel | Скрипты миграции из старого IDE |
| **5** | Интеграционное тестирование | Финальные правки | Деплой и автообновление |

### 4.2 Архитектурные улучшения

1. **Capability-based routing** — Engine маршрутизирует по capability, не по имени агента
2. **Plugin-based алгоритмы** — 5 алгоритмов как plugin registry, а не фиксированный список
3. **MCP/OpenAPI для Desktop↔Engine** — не кастомный WebSocket
4. **Контракты перед реализацией** — сначала API-схемы, потом код
5. **Добавить разделы:** Безопасность канала, Деплой, Тестирование, Degraded-mode

### 4.3 Ключевое решение для Недели 1

> **Неделя 1 = Engine API контракт + скелет Desktop. Ничего больше.**

Не делать Agent Dashboard и HomeCanvas, пока Engine не готов. Сначала API-контракт, потом UI.
